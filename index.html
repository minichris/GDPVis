Features

  • Clicking on a node will change the currently viewed article.
  • Hovering over a node will show its name.
  • The graph can be zoomed using the mouse wheel.
  • The nodes can be tugged about by click-dragging a node.
  • The graph can be panned by clicking and dragging the background.
  • Clicking the "Filters" button will display the currently enabled filters.
  • In the filters menu, the filters are appiled in order, top to bottom. There is currently no way to do "OR" logic.
  • The filters only update when you press "Apply Filters".
  • Not applying enough filtering will cause a message to come up warning you that you are trying to display too many nodes. It is highly recomended you heed this warning.

Planned

  • Currently, the visualization has example patterns of patterns found in the game "World of Warcraft" and patterns in the Negative Patterns category.
  • Currently, the nodes on the graph have random colours and links have random strengths.
  • Currently, the graph shows all links found in the article. Eventually, it may only show links found in the "Relations" segment of every pattern article.
  • Currently, hovering over the links between nodes shows no tooltip. Eventually, this will show the context of the link from the article.
  • The browser is currently implemented as an iframe, eventually it might be changed to a different method (to better fit the theming and reduce clutter).
  • Currently, there is no way to save your filter set-up.
  • Working title!
